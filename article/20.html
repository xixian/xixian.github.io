常见博客程序

折腾过的，简单闲聊一下

大概有：
xlog
GitHub Pages
Gitee Pages
Gridea
Halo
Hexo
Hugo
WordPress
Gitbook
VuePress
VitePress
Docusaurus
Docsify
Dumi
Typecho
大概可以分成三类

#????1. 静态，页面#

i. Github Pages#

国内不开代理访问慢，可能会进不去

支持免费绑定自己的域名

QQ，微信内可以直接打开



ii. Gitee Pages#

国内访问快

不支持免费绑定自己的域名

QQ，微信内可以直接打开


iii. Gridea#

静态博客写作客户端，markdown 语法

主题较少

写好的文章可以直接同步至 GitHub/Gitee 仓库

通过→Pages 部署即可访问



????2. 三 H / 一 W#

i. Halo#

简易，快速
所写，所得
搭配服务器
Markdown 语法
宝塔/1panel可视化管理面板都支持一键部署
推荐使用 1panel



ii. Hexo#

快速、简洁、高效
易上手，多主题
可部署在 Github/Gitee
Markdown 语法


iii. Hugo#

速度和灵活性
多主题
Markdown 语法
用的比较少

iv. WordPress#

流行的建站之一
多主题，多插件
可视化面板控制
满足大部分站点
比较吃服务器配置
推荐 2H2G


#????3. 文档 / 小站 / 博客 / 技术栈#

i. Gitbook#

Markdown 语法
免费自定义域名
本地版已不在更新
网页版不建议对国内用户
简洁，高效


ii. VuePress#

个人知识库
简洁至上
Vue 驱动，高性能
多主题多插件
支持 vercel 部署
Markdown 语法


iii. VitePress#

Vite 通常比 VuePress 快
特别适用于文档网站
Vite 生态系统
插件和主题生态系统少
支持 vercel 部署
Markdown 语法


iv. Docusaurus#

无需深入的前端开发知识
易于使用
内置搜索功能，支持动画
支持插件系统
活跃的开发社区
支持 vercel 部署
Markdown 语法
如果你需要好看的动画效果，非常推荐你使用 docusaurus


v. 技术栈#

Docsify
Dumi
轻量文档网站：推荐 Docsify
复杂库的文档：推荐 Dumi
vi. Typecho#

轻量化
友好，易上手
多简洁主题
支持 vercel 部署
Markdown 语法
5F9F45304D0C9C8D5A94878973D22531

????最后#

1. 部署至 Github/Gitee

免费，无需担心攻击
适合个人博客，静态页面
2. 部署至Vercel

免费容器，一键部署
国内可不开代理访问
无需担心攻击
适合文档网站
3. 部署至服务器

云服务器商
阿里云
腾讯云
域名购买
万网
西部数码
域名备案
域名解析
了解 CDN
服务器防护
了解 Linux 发行版
Ubuntu（推荐）
Debian
CentOS
如果使用海外服务器（无需备案）
